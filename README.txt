HOW TO USE:

1. Plug in a USB stick (min 4 GB).
2. Run build-usb.bat as Administrator.
3. Enter the drive letter (like E) when prompted.
4. USB will be flashed with Debian 12 NetInstall and preseed.cfg copied.
5. Boot your target machine from USB. Installation is 100% automated.
6. Home Assistant will run at http://your-home-ip:8123
7. ZoneMinder at http://your-home-ip/zm
